# DSA-Stack-LinkedList-CPP
Implementation of Stack Data Structure using Linked List in C++. Part of my DSA learning journey.
# Stack Implementation using Linked List

This project demonstrates the **Stack Data Structure** implemented with a dynamic **Linked List** in C++.

#Features
- **Push:** Adds an element to the top of the stack.
- **Pop:** Removes the top element (with underflow check).
- **Display:** Prints all elements in the stack.
- **Dynamic Memory:** Uses pointers and nodes for efficient memory allocation.

# Concepts Used
- Object-Oriented Programming (Classes & Objects)
- Dynamic Memory Management (`new` and `delete`)
- Pointers and Linked Nodes

#How to Run
1. Clone this repository.
2. Compile using: `g++ Stack_LinkedList.cpp -o main`
3. Run: `./main`
